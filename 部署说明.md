# 部署到 GitHub Pages 并安装为手机 PWA

## 文件说明
- `index.html` — 主程序（已加入 PWA 支持代码）
- `manifest.json` — PWA 配置文件（应用名称、图标、启动方式）
- `service-worker.js` — 离线缓存脚本
- `icon-192.png` / `icon-512.png` — 应用图标（占位图标，可自行替换）

## 第一步：创建 GitHub 仓库
1. 登录 GitHub，点击右上角 `+` → `New repository`
2. 仓库名随意，例如 `qhyd-sim`，设置为 `Public`（GitHub Pages 免费版需要公开仓库）
3. 不要勾选"添加 README"，直接创建

## 第二步：上传文件
有两种方式，选一种即可：

**方式A：网页拖拽上传（最简单）**
1. 进入新建好的仓库页面，点击 `Add file` → `Upload files`
2. 把本压缩包里的 5 个文件（`index.html`、`manifest.json`、`service-worker.js`、`icon-192.png`、`icon-512.png`）全部拖进去
3. 点击下方 `Commit changes`

**方式B：命令行上传**
```bash
git clone https://github.com/你的用户名/qhyd-sim.git
cd qhyd-sim
# 把这5个文件复制进来
git add .
git commit -m "init pwa"
git push
```

## 第三步：开启 GitHub Pages
1. 进入仓库 `Settings` → 左侧菜单 `Pages`
2. `Source` 选择 `Deploy from a branch`
3. `Branch` 选择 `main`，目录选择 `/ (root)`，点击 `Save`
4. 等待 1-2 分钟，页面会显示一个网址，形如：
   `https://你的用户名.github.io/qhyd-sim/`

这个网址就是你以后用手机访问的地址。

## 第四步：手机上安装为 PWA

**iPhone (Safari)：**
1. 用 Safari 打开上面的网址（注意必须用 Safari，微信内置浏览器不支持）
2. 点击底部分享图标（方框+箭头）
3. 选择"添加到主屏幕"
4. 桌面上会出现一个独立图标，点击即可像 App 一样全屏打开

**Android (Chrome)：**
1. 用 Chrome 打开网址
2. 点击右上角菜单（三个点）
3. 选择"安装应用"或"添加到主屏幕"
4. 安装后会出现在应用列表/桌面

## 注意事项
- GitHub Pages 必须使用 **HTTPS**，PWA 的 Service Worker 才能正常工作，GitHub Pages 默认就是 HTTPS，无需额外配置。
- 页面里仍然通过 `https://cdn.jsdelivr.net` 加载 Chart.js，所以**首次打开需要联网**；之后由于浏览器和 Service Worker 的缓存，部分功能可离线使用，但图表库本身建议保持联网以确保稳定加载。
- 如果以后修改了 `index.html`，记得同时把 `service-worker.js` 里的 `CACHE_NAME` 改个新版本号（比如 `qhyd-sim-v2`），否则手机上可能因为缓存而看不到更新，需要在浏览器里手动刷新或重新安装。
- 想换图标的话，直接用同名文件替换 `icon-192.png` 和 `icon-512.png` 再上传即可，建议用正方形图片。
